# FLUX.1-Dev Validated Vocabulary
## Terms with HIGH reliability in image generation

**Source:** Community research, prompt datasets, production testing
**Constraint:** Only use terms from this file in generated prompts

---

## SHOT TYPES (HIGH RELIABILITY)

### Portrait/Character Framing
| Term | Captures | Use Case |
|------|----------|----------|
| `extreme close-up` | Eyes, small detail | Evidence, emotional eyes |
| `close-up shot` | Face fills frame | Reactions, expressions |
| `intimate close-up shot` | Face with intense focus | Emotional beats |
| `medium close-up` | Head and shoulders | Dialogue scenes |
| `medium shot` | Waist-up | Standard character shot |
| `upper-body portrait` | Chest up | Character showcase |
| `cowboy shot` | Mid-thigh up | Action poses with weapons |
| `medium full shot` | Knees up | Character with gesture |
| `full body shot` | Head to toe | Full character reveal |
| `complete figure shot` | Entire body with space | Environmental context |
| `standing full body shot` | Full standing figure | Character establishment |

### Scene Framing
| Term | Captures | Use Case |
|------|----------|----------|
| `wide shot` | Full body + environment | Location establishment |
| `extreme wide shot` | Vast environment, small figure | Scale, isolation |
| `establishing shot` | Location-focused | Scene openers |

### Specialty Shots
| Term | Captures | Use Case |
|------|----------|----------|
| `macro shot` | Extreme detail on object | Evidence, technology |
| `forensic shot` | Clinical documentation style | Investigation scenes |
| `silhouette shot` | Figure against bright background | Dramatic reveals |
| `portrait photograph` | Professional portrait style | Character headshots |

## CAMERA ANGLES (MEDIUM-HIGH RELIABILITY)

### Vertical Angles
| Term | Effect | Emotional Use |
|------|--------|---------------|
| `eye-level shot` | Neutral, natural | Standard scenes |
| `low angle shot` | Subject appears powerful | Antagonists, power |
| `high angle shot` | Subject appears vulnerable | Danger, overview |
| `worm's-eye view` | Extreme low angle | Epic scale |
| `bird's-eye view` | Directly overhead | Tactical overview |
| `overhead shot` | Looking straight down | Maps, evidence layout |
| `Dutch angle` | Unease, disorientation | Horror, tension |

### Horizontal Positioning
| Term | Effect |
|------|--------|
| `front view` | Subject facing camera |
| `side view` | 90 degrees from camera |
| `profile` | Side view (synonym) |
| `three-quarter view` | 45 degrees from camera |
| `back view` | Subject facing away |

## TERMS TO AVOID

| Term | Issue |
|------|-------|
| `tracking shot` | Video term, static images only |
| `dolly zoom` | Video term |
| `pan` / `tilt` | Camera movement |
| `white background` | Causes blur in FLUX |
| `(word:1.2)` syntax | SD weight syntax, not supported |
